# ひなテックGames プロジェクト指示書（Claude Code用）

## プロジェクト概要

プログラミング教室「ひなテック」が運営するブラウザゲームサイトを構築する。
子どもから大人まで楽しめるカジュアルゲームを提供し、プログラミング教室への自然な送客を実現する。

- **サイト名**: ひなテックGames
- **URL**: games.hinata-ya.tech（サブドメイン）
- **ホスティング**: ConoHa（既存WPサイト hinata-ya.tech と同居）
- **技術構成**: 静的HTML / CSS / JavaScript（フレームワーク不使用、ビルド不要）
- **デプロイ**: SFTP でファイルをアップロード

- 各ページには `<div id="header"></div>` `<div id="footer"></div>` のプレースホルダーを配置

## デザイン方針

### 全体トーン

- **明るく親しみやすい**デザイン。子どもが見ても楽しそうと思える雰囲気
- 色使いはひなテックのブランドカラーがあればそれに合わせる。なければ、暖色系（オレンジ・イエロー）をメインに
- フォントはシステムフォント（Noto Sans JP, sans-serif 等）でOK
- 角丸・やわらかい影を使ったカード型UIで統一

### レスポンシブ対応

- **スマホファースト**で設計
- ブレークポイント: 768px（タブレット）、1024px（PC）
- ゲームは縦画面スマホでも操作可能にすること

### トップページ構成

1. **ヘッダー**: ロゴ + サイト名「ひなテックGames」+ ナビゲーション
2. **ヒーローエリア**: キャッチコピー「遊んで学ぼう！ブラウザで遊べる無料ゲーム」
3. **ゲーム一覧**: カード型のグリッドレイアウト（サムネ + ゲーム名 + カテゴリタグ）
4. **ひなテック紹介バナー**: 「このゲーム、自分でも作れるよ！→ ひなテックで学ぶ」
5. **フッター**: コピーライト + ひなテック本体サイトへのリンク

### ゲームページ構成

1. **ヘッダー**（共通）
2. **ゲームタイトル + カテゴリタグ**
3. **ゲームエリア**（Canvas等）: 画面中央に配置、最大幅制限あり
4. **操作説明**: ゲームの下にシンプルに表示
5. **スコア表示**: ゲーム内 UI として実装
6. **「このゲームを作ってみよう！」セクション**: ゲームの仕組みを簡単に解説 + ひなテックへの誘導リンク
7. **他のゲームへの導線**: 「他のゲームも遊ぶ」リンク
8. **フッター**（共通）

## 最初のゲーム：2048

### ゲーム仕様

- **ルール**: 4×4のグリッドで同じ数字のタイルをスライドして合体させ、2048を目指す
- **操作方法**:
  - PC: 矢印キーまたは WASD
  - スマホ: スワイプ操作（上下左右）
- **UI要素**:
  - 現在のスコア表示
  - ベストスコア表示（localStorage に保存）
  - 「New Game」ボタン
  - ゲームオーバー時のオーバーレイ表示（スコア + リトライボタン）
  - 2048達成時の「You Win!」表示（続行も可能に）
- **アニメーション**:
  - タイル出現時のポップアニメーション
  - タイル移動時のスライドアニメーション
  - タイル合体時の拡大縮小アニメーション
- **タイルの色**:
  - 数字ごとに異なる背景色（2=薄いベージュ、4=やや濃いベージュ、8=オレンジ、16=濃いオレンジ、32=赤系、64=赤、128=黄色、256=黄色、512=金色、1024=金色、2048=金色光る）
- **レスポンシブ**:
  - スマホ: グリッドが画面幅に収まるようにサイズ調整
  - ゲーム領域外のスワイプでページスクロールが発生しないように制御

### 技術要件

- **Canvasは不使用**: CSSグリッド + div で実装（DOMベース）
- **アニメーション**: CSS transition / animation で実装
- **タッチ操作**: touchstart / touchend イベントでスワイプ方向を判定
- **スコア保存**: localStorage 使用
- **外部ライブラリ不使用**: Vanilla JS のみ

## SEO対策

- 各ページに適切な `<title>` と `<meta description>` を設定
  - トップ: 「ひなテックGames｜ブラウザで遊べる無料ゲーム」
  - 2048: 「2048パズル｜無料ブラウザゲーム｜ひなテックGames」
- OGP タグ設定（SNSシェア用）
- `<html lang="ja">` を設定
- 構造化データ（JSON-LD）で WebSite + Game を記述
- sitemap.xml を設置
