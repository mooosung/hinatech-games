ひなテックについて

プログラミング教室「ひなテック」

ひなテックは、東京都国分寺市にあるプログラミング教室です。
子どもから大人まで、ゲームやWebサイトの制作を通して、
楽しくプログラミングを学ぶことができます。

学べる内容

  • Scratch
  • HTML / CSS でWebサイト制作
  • JavaScript でゲーム開発
  • 3Dプリンタ
  • ジュニアプログラミング検定
  • プログラミングの基礎的な考え方
  • 作品を形にするものづくりの楽しさ

教室情報

  • 場所: 東京都国分寺市
  • 対象: 小学生〜大人まで
  • スタイル: 少人数制で一人ひとりに合わせた指導

このサイトのゲームも教材です

ひなテックGamesで遊べるゲームは、すべて HTML・CSS・JavaScript で作られています。 教室では、こうしたゲームの仕組みを理解しながら、自分の手でゲームを作る体験ができます。

詳しくは教室サイトをご覧ください
