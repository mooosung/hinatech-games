遊び方

  • カードをクリック（タップ）してめくります
  • 2枚めくって同じ絵柄ならペア成立！ カードはそのまま表になります
  • 絵柄が違ったら、カードは裏に戻ります
  • すべてのペアを見つけたらクリア！
  • 少ない手数でクリアするほど高評価（星3つが最高）
  • 難易度を変えてチャレンジしよう！
